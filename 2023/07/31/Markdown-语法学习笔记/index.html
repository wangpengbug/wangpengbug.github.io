Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown 标题

使用 = 和 - 标记一级和二级标题

使用 # 号标记

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

Typora快捷键

1-6 级标题：Ctrl+1~6

段落：Ctrl+0

提升标题级别：Ctrl+=

降低标题级别：Ctrl+-

Typora软件上点击对应板块即可查看，其他快捷键不做赘述。

Markdown 段落

换行

段落的换行是使用两个以上空格加上回车(我直接回车）。

段落后面使用一空行来表示重新开始一个段落。

分隔线

三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。（可以在星号或是减号中间插入空格。）

删除线

Markdown 列表

无序列表

星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格

  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项

有序列表

数字并加上 . 号

  1. 第一项
  2. 第二项
  3. 第三项

列表嵌套

在子列表中的选项前面添加两个或四个空格

  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素

Markdown 区块

在段落开头使用 > 符号 ，然后后面紧跟一个空格(区块是可以嵌套的,可以配合列表嵌套)

Markdown 代码

一个函数或片段的代码

用反引号把它包起来（`）

main() 函数

代码区块

使用 4 个空格或者一个制表符（Tab 键）。
用 ```包裹一段代码，并指定一种语言（也可以不指定）

Markdown 图片

图片语法格式

  • 开头一个感叹号 !
  • 接着一个方括号，里面放上图片的替代文字
  • 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。

像网址那样对图片网址使用变量:

然后在文档的结尾为变量赋值（网址）

指定图片高度与宽度，使用普通的<img>标签。

使用|分隔不同的单元格（列）

Markdown 高级技巧

支持的 HTML 元素

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。

目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等

使用 Ctrl+Alt+Del 重启电脑(一些原因，在此没显示出效果。）

转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，
